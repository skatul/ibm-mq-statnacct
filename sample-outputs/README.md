# Sample Outputs

This directory contains example JSON output files generated by the IBM MQ Stats Reader.

## Files

- `mq_stats_20251108_075733.json` - Sample statistics output from MQQM1 queue manager
- `mq_stats_20251108_080240.json` - Another sample with different timestamp
- `mq_stats_sample_demo_20251108_080324.json` - Demo output showing expected format

## JSON Structure

The output files follow this structure:
```json
{
  "collection_timestamp": "2024-11-08T12:00:00Z",
  "queue_manager": "MQQM1",
  "statistics": [
    {
      "timestamp": "2024-11-08T12:00:00Z",
      "queue_name": "APP1.REQ",
      "get_count": 10,
      "put_count": 5,
      "open_input_count": 2,
      "open_output_count": 1
    }
  ],
  "summary": {
    "total_queues": 2,
    "total_gets": 25,
    "total_puts": 15,
    "active_readers": 3,
    "active_writers": 4
  }
}
```

## Usage

These sample files can be used for:
- Testing time series database integrations
- Understanding the expected output format
- Validating JSON parsing in downstream applications
- Documentation and examples
