# Sample Outputs

This directory contains example JSON output files generated by the IBM MQ Stats Reader with enhanced PCF parsing.

## Files

- `mq_stats_20251108_082716.json` - Sample statistics output from MQQM1 queue manager
- `mq_stats_20251108_082908.json` - Enhanced output with improved PCF parsing
- `complete_app_output_20251119_205404.json` - Complete application output with clean parameter parsing

## Enhanced Output Features

**November 2025 Updates**:
- ✅ Clean parameter parsing (no more UNKNOWN_PARAM entries)
- ✅ Proper STAT (21) and ACCOUNTING (25) message type identification
- ✅ Enhanced producer/consumer detection
- ✅ Client IP and application name extraction
- ✅ Graceful handling of corrupted PCF data

## JSON Structure

The output files follow this structure:
```json
{
  "collection_timestamp": "2024-11-08T12:00:00Z",
  "queue_manager": "MQQM1",
  "statistics": [
    {
      "timestamp": "2024-11-08T12:00:00Z",
      "queue_name": "APP1.REQ",
      "get_count": 10,
      "put_count": 5,
      "open_input_count": 2,
      "open_output_count": 1
    }
  ],
  "summary": {
    "total_queues": 2,
    "total_gets": 25,
    "total_puts": 15,
    "active_readers": 3,
    "active_writers": 4
  }
}
```
